# cv-typst 🏢
`Typst` template for Curriculum-Vitae.

## Installation and Usage 🚀

### Method 1: Typst Template

To install the template, simply run the following commands in your terminal:

```
mkdir -p ~/.local/share/typst/packages/local
git clone git@github.com:Alvinceleste2/cv-typst.git ~/.local/share/typst/packages/local/cv
```

Once the template is installed, you can initialize a new document from it by running:

```
typst init @local/cv <folder_name>
```

where `folder_name` is the name you want to give to the folder that will contain the document.

### Method 2: Direct Clone
You can also download the latest release of the template and start editing the provided `cv-typst-template.zip` file.
